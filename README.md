# url-shortner
url-shortner uygulaması kullanıcıdan aldığı geçerli bir url'i daha kısa bir url haline getiren bir uygulamadır. 



## Kullanım 

### 1. Http://localhost:3000/ adresine kısaltmak isteğimiz url'i post isteği olarak gönderilir. Bu istek sonucunda kısa olarak kısaltımış url döner. Aşağıda görüldüğü gibi

![Url kısaltma isteği](https://github.com/bunyamintunc/url-shortner/blob/master/images/post-long-url.png)

### 2. Girilen url'in geçersiz olması durumunda "url geçerli değil hata mesajı döner".

![Geçersiz url](https://github.com/bunyamintunc/url-shortner/blob/master/images/invalid-url.png)

### 3. Girilen url hali hazırda mevcut ise girilen url'in zaten var olduğu mesajı döner. 

![Var olan url](https://github.com/bunyamintunc/url-shortner/blob/master/images/url-exist.png)

### 4. Kısaltılmış url ile tarayıcıdan istek yapılır ve url'in işaret ettiği sayfaya yönlendirme yapılır.

![Yönlendirme](https://github.com/bunyamintunc/url-shortner/blob/master/images/short-url.png)

### 5. Kısaltılmış olan url'in işaret ettiği sayfa açılır.

![Sayfa açılışı](https://github.com/bunyamintunc/url-shortner/blob/master/images/redirect-url.png)
